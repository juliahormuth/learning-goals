# Como o computador funciona? 

## CPU - Central Processing Unit
- Os mais sofisticados possuem mais de um único núcleo (core).

- O core é quem realiza acoes de adicao, subtracao, multiplicacao, divisao, carregamento e salvando informacoes na memória.

- Ex: Com 2 cores, ele consegue fazer 2 tarefas simultaneamente.

## RAM - Random Access Memory
- Armazena seus dados enquanto o pc está ligado/funcionando. É volátil.
- Armazena até 16GB, por exemplo.

## HARD DISK
- Dados ficam armazenados mesmo com o pc desligado.
- Armazena até 265GB, 1TB e etc.
- Energia é usada apenas para ler ou alterar dados.
