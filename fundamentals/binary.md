# O que o computador entende?

## Mundo real - dezena, centena e etc
```
100 10 1
1    2 3
100x1+2x10+3
100+20+3
123
```

## Mundo dos computadores - Binário 
```
4   2   1
0   0   0
4x0+2x0+1x0
0+0+0
0

```

- `000` representa `0` no mundo real.

```
4   2   1
1   1   1
4x1+2x1+1x1
4+2+1
7
```

- `111` em binário é igual a `7`

- Com 8 já é necessário um Bit a mais:

```
8 4 2 1
1 0 0 0
8x1+4x0+2x0+1x0
8+0+0+0
8
```

- `1000` em binário é igual a `8`.

- **Exercício das lampadas, para o número 8 seria necessário 4 lampadas ao invés de 3.**

- **Um conjunto de 8 bits é um Byte**

## Tabela ASCII
É uma tabela de números decimais para letras. Ex: 65 = Letra A.
